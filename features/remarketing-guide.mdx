---
title: "How to Run Remarketing Campaigns Using Linkrunner"
icon: "rocket"
---

Follow these steps to set up and run remarketing campaigns with Linkrunner.

<Steps>
  <Step title="Understand how reattribution works">
    Learn how Linkrunner detects returning users, the different reactivation events, and how attribution is assigned.

    [Remarketing & Reattribution](/features/remarketing)
  </Step>
  <Step title="Set up deep link capture in your SDK">
    Call the `handleDeeplink` function in your SDK to capture deep link opens. This is required for Linkrunner to detect returning users via deep links and trigger reattribution.

    - [Android](/sdk/android#handle-deeplink)
    - [iOS](/sdk/ios#handle-deeplink)
    - [React Native](/sdk/react-native#handle-deeplink)
    - [Flutter](/sdk/flutter#handle-deeplink)
    - [Capacitor](/sdk/capacitor#handle-deeplink)
  </Step>
  <Step title="Create audience segments">
    Build custom audience segments on Meta and Google using tracked events already being sent to the ad platform.

    [Creating Audience Segments](/features/remarketing-audience-segments)
  </Step>
  <Step title="Launch your remarketing campaign">
    Use the audience segments you created to run remarketing campaigns on Meta and Google.

    [Running Remarketing Campaigns](/features/remarketing-campaigns)
  </Step>
  <Step title="Configure reattribution windows">
    Set the inactivity window and lookback window from the Linkrunner dashboard to control how returning users are attributed.

    [Dashboard > Settings > Remarketing](https://dashboard.linkrunner.io/settings)
  </Step>
</Steps>
