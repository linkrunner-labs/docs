## Core Features

-   **Custom Domain Links** - Branded links using your own domain
-   **Multi-Platform Attribution** - Track users from click to revenue across web and mobile
-   **Deferred Deep Linking** - Direct users to specific in-app content after install
-   **Revenue Attribution** - Connect revenue to acquisition source
-   **Multi-Cloud Deployment** - Low-latency tracking from US, EU, and Asia
