Welcome to Linkrunner's Feature Releases documentation. This is your central hub for all product updates, new features, and improvements to the Linkrunner platform.

## What's New

We continuously improve Linkrunner to provide you with the best attribution and marketing analytics tools. The Feature Releases section is organized to help you quickly find information about:

- **Latest Features**: Recently deployed capabilities and enhancements
- **Bug Fixes**: Important issues we've addressed
- **Performance Improvements**: Updates that make Linkrunner faster and more reliable
- **SDK Updates**: New versions and capabilities for our integration SDKs

## Release Cycle

Linkrunner follows a regular release cycle with:

- **Major Releases**: Quarterly updates with significant new features
- **Minor Releases**: Monthly updates with improvements and smaller features
- **Patch Releases**: As-needed fixes for critical issues

Each release is thoroughly tested to ensure a seamless experience for your marketing analytics workflows.
